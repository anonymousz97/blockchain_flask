# blockchain_flask

Requirements : 

- flask
- tqdm

Run : 
- Extract file rar.
WINDOWS : (cmd)

- Run `set FLASK_APP = flaskr`

- Run `set FLASK_ENV = development`

LINUX : (terminal)

`$ export FLASK_APP=flaskr`

`$ export FLASK_ENV=development`

`$ flask run`

- Run flask run
Rerun database : 
- `flask init-db`
