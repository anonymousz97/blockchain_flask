# blockchain_flask

Requirements : 

- flask
- tqdm

Cách chạy : 
- Extract file rar.
WINDOWS : (cmd)
- Chạy set FLASK_APP = flaskr
- Chạy set FLASK_ENV = development
LINUX : (terminal)
$ export FLASK_APP=flaskr
$ export FLASK_ENV=development
$ flask run

- Chạy flask run
Chạy lại database : 
- flask init-db
